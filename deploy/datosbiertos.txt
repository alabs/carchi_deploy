# Deploy Datos Abiertos

1. Para recargar el código hace falta descargar el código del tema, ver en https://github.com/alabs/ckanext-carchi_theme - Si se quiere comprobar en local se puede ver la instalación en este mismo repositorio, direcotorio install/

2. Se suben los cambios a la rama master

3. Se conecta al servidor que corresponda (stag-datosabiertos o prod-datosabiertos).

```
source /usr/lib/ckan/default/bin/activate
pip install --upgrade --no-deps --force-reinstall https://github.com/alabs/ckanext-carchi_theme/zipball/master 
sudo service apache2 reload
```
